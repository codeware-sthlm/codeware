# Code & CopyButton

Syntax-highlighted code blocks powered by [prism-react-renderer](https://github.com/FormidableLabs/prism-react-renderer). `Code` renders line numbers and embeds a `CopyButton` in the top-right corner automatically.

```tsx
import { Code } from '@codeware/shared/ui/code';
import { CopyButton } from '@codeware/shared/ui/copy-button';
```
