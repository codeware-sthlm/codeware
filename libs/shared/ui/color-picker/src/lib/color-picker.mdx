# ColorPicker

A searchable Tailwind color palette picker backed by Radix UI Popover + Command. The selected value is a Tailwind color name like `"teal-500"`.

```tsx
import { ColorPicker } from '@codeware/shared/ui/color-picker';
```

The component wraps itself in `.twp` so it works correctly inside Payload Admin without an extra wrapper.
