# IconPicker

A searchable Heroicons grid picker backed by Radix UI Dialog. The selected value is a Heroicon component name like `"StarIcon"` (from `@heroicons/react/20/solid`).

```tsx
import { IconPicker } from '@codeware/shared/ui/icon-picker';
```

The component wraps itself in `.twp` so it works correctly inside Payload Admin without an extra wrapper.
