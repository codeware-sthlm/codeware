# Primitives

Low-level building blocks for UI components.
These are not meant to be used directly in most cases, but they can be useful for custom components that need a specific behaviour or style.

## Callout

Inline contextual messages. Built on top of `Alert` from shadcn/ui. Supports `info`, `warning`, and `tip` kinds — each maps to a distinct Heroicon automatically.

## InlineIcon

Renders a sized icon from either raw SVG markup or an image URL. Pass exactly one of `svgCode` or `src`.

SVG markup with `fill="currentColor"` inherits the surrounding CSS color, making it trivial to tint in context. The image URL path emits a plain `<img>` — no framework-specific optimisation.

```tsx
// SVG source — inherits currentColor
<InlineIcon svgCode={myLogoSvg} size={48} />

// Image URL source
<InlineIcon src="https://example.com/logo.png" size={48} />
```
