# Alert & Tooltip

## Alert

Inline feedback with optional icon, title, and description. Use `variant="destructive"` for errors.

```tsx
import { Alert, AlertTitle, AlertDescription } from '@codeware/shared/ui/shadcn/components/alert';
```

## Tooltip

Wraps any trigger. `TooltipProvider` is built into the `Tooltip` component — no separate provider needed.

```tsx
import { Tooltip, TooltipTrigger, TooltipContent } from '@codeware/shared/ui/shadcn/components/tooltip';
```
