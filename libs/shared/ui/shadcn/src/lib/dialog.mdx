# Dialog

Modal overlay built on Radix UI Dialog. Content renders in a portal outside the normal DOM tree.

```tsx
import {
  Dialog, DialogTrigger, DialogContent,
  DialogHeader, DialogTitle, DialogDescription, DialogFooter
} from '@codeware/shared/ui/shadcn/components/dialog';
```

**Payload Admin note:** dialog content renders via a portal and loses `.twp` ancestry. The CMS `custom.css` handles this explicitly with `div[role='dialog'][data-slot='dialog-content']` — see the Introduction for details.
